# DS18B20 Sensor Setup

## Wiring

Connect the DS18B20 data pin to GPIO4 (pin 7) with a 4.7k pull-up resistor to 3.3V.

## Enable One-Wire Interface

```bash
# Add to /boot/config.txt
dtoverlay=w1-gpio

# Reboot
sudo reboot
```

## Verify Detection

```bash
ls /sys/bus/w1/devices/28-*
```

## Enable in PiMon

Add to `.env`:

```
SENSOR_DS18B20_ENABLED=true
```

If your one-wire devices are at a non-standard path:

```
DS18B20_BASE_DIR=/sys/bus/w1/devices
```
